Machine Learning Fundamentals

Machine learning is a subset of artificial intelligence that enables systems to learn and improve from experience without being explicitly programmed. It focuses on developing algorithms that can access data and use it to learn for themselves.

Types of Machine Learning:
1. Supervised Learning - Learning from labeled data to make predictions
2. Unsupervised Learning - Finding patterns in unlabeled data
3. Reinforcement Learning - Learning through trial and error with rewards

Key Concepts:
- Training Data: The dataset used to train the model
- Features: Input variables used for predictions
- Labels: Target values in supervised learning
- Model: The mathematical representation learned from data
- Overfitting: When a model performs well on training data but poorly on new data
- Underfitting: When a model is too simple to capture patterns in data

Common Algorithms:
- Linear Regression: For continuous output prediction
- Decision Trees: For classification and regression
- Neural Networks: For complex pattern recognition
- Support Vector Machines: For classification tasks
- K-Nearest Neighbors: For classification based on proximity
